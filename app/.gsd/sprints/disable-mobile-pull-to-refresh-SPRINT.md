# Sprint 1 — Disable Mobile Pull-to-Refresh

> **Duration**: 2026-04-27 to 2026-04-27
> **Status**: In Progress

## Goal
Impedir el refresco de página por deslizamiento (pull-to-refresh) en dispositivos móviles para mejorar la experiencia de usuario tipo App.

## Scope

### Included
- Agregar `overscroll-behavior-y: none` en el CSS global (`index.css`).
- Asegurar que el cambio afecte tanto a `html` como a `body`.

### Explicitly Excluded
- Cambios en la lógica de recarga de datos interna.
- Rediseño de componentes de scroll.
