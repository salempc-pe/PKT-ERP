# Plan 23.3: Módulo de Gestión de Equipo en Cliente

## Objective
Crear el módulo `TeamModule.jsx` para que el usuario Client (con rol Admin de su tenant) pueda invitar a nuevos colaboradores a su organización.

## Context
- .gsd/ROADMAP.md
- src/layouts/ClientLayout.jsx
- src/modules/client/team/TeamModule.jsx (nuevo)

## Tasks

<task type="auto">
  <name>Crear el módulo TeamModule y conectarlo</name>
  <files>
    - src/modules/client/team/TeamModule.jsx
    - src/layouts/ClientLayout.jsx
    - src/App.jsx
  </files>
  <action>
    - Crear `TeamModule.jsx`:
      - Debe listar los usuarios pertenecientes al `user.organizationId` (usando un hook mock o directamente de `getClientUsers()`).
      - Debe incluir un botón de "Invitar Miembro" y un formulario simple para nombre, email y rol (`user` o `admin`).
    - En `ClientLayout.jsx`:
      - Añadir el enlace al Sidebar bajo una sección "Configuración" o junto al perfil: `<Link to="/client/team">Equipo</Link>`, pero validando que el `user.role === 'admin'`. (Si el rol no es admin, no se muestra el link).
    - En `App.jsx`, añadir la ruta `/client/team` apuntando a `TeamModule`.
  </action>
  <verify>grep -A 5 "/client/team" src/App.jsx && grep "TeamModule" src/layouts/ClientLayout.jsx</verify>
