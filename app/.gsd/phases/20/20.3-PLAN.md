# Plan 20.3: Modales de Facturación e Inventario

## Objective
Al marcar una factura/boleta como "Pagada", debe aparecer un modal básico mostrando una opción de previsualización para "Imprimir". Además, el modal de Inventario no debe oscurecer por completo el fondo; se debe asegurar que se visualice la lista del inventario y la app detrás (efecto overlay/backdrop semi-translúcido), replicando visualmente el z-index/backdrop del modal de "Emitir Factura".

## Context
- .gsd/ROADMAP.md
- app/src/modules/client/sales/SalesModule.jsx
- app/src/modules/client/inventory/InventoryModule.jsx
