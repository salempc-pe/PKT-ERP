# Plan 6.1: Capa de Datos (useFinance)

## Objective
Implementar el núcleo de gestión de datos para el módulo de Finanzas utilizando Firebase Firestore con soporte para operaciones de fallback mock. El objetivo es poder registrar ingresos y gastos asociados a un Tenant (orgId).

- .gsd/phases/6/RESEARCH.md
