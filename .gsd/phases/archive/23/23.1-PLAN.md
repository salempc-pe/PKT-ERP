# Plan 23.1: Actualizar `AdminOrganizations` para `maxUsers`

## Objective
Permitir que el Super Admin defina y actualice el campo `maxUsers` (límite de usuarios contratados) en la información y edición de cada organización.

## Context
- .gsd/ROADMAP.md
- src/modules/admin/clients/AdminClients.jsx
- src/context/AuthContext.jsx

## Tasks

<task type="auto">
  <name>Agregar campo maxUsers a AdminClients (Nueva Org)</name>
  <files>
    - src/modules/admin/clients/AdminClients.jsx
    - src/context/AuthContext.jsx
  </files>
  <action>
    En `AdminClients.jsx`:
    - En el estado `newOrgData`, añadir la propiedad `maxUsers: 5`.
    - En el formulario modal de "Nueva Organización", agregar un input numérico para definir el "Límite de Usuarios (maxUsers)".
    - Mostrar el `maxUsers` actual en la UI de cada tarjeta de empresa (por ejemplo: "Usuarios ({org.users.length}/{org.maxUsers || 5})").
    - En `AuthContext.jsx`, asegurar que al crear empresa (`adminCreateOrg`), se guarda `maxUsers`.
  </action>
  <verify>grep -A 5 "maxUsers" src/modules/admin/clients/AdminClients.jsx</verify>
  <done>El campo maxUsers se captura al crear una nueva empresa y se muestra en la UI.</done>
</task>

<task type="auto">
  <name>Agregar campo maxUsers a AdminClients (Edición de Org)</name>
  <files>
    - src/modules/admin/clients/AdminClients.jsx
    - src/context/AuthContext.jsx
  </files>
  <action>
    En `AdminClients.jsx`:
    - En el estado `editOrgData`, añadir la propiedad `maxUsers`.
    - En `handleOpenEditOrg`, capturar `maxUsers` del org (default a 5 si no existe).
    - En el modal de "Editar Organización", agregar el input para cambiar "Límite de Usuarios".
    - Guardar cambios a través de `handleSaveOrgDetails`.
  </action>
  <verify>grep -A 5 "Administrar límite de usuarios" src/modules/admin/clients/AdminClients.jsx || grep -i "maxUsers" src/modules/admin/clients/AdminClients.jsx</verify>
  <done>Se puede modificar el límite de usuarios de una empresa existente.</done>
</task>

## Success Criteria
- [ ] La UI de AdminClients permite definir `maxUsers` al crear y editar organizaciones.
- [ ] La tarjeta de cada organización muestra cuántos usuarios tiene frente al máximo.
