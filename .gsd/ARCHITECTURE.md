## Overview

PKT ERP es una plataforma empresarial modular construida sobre React y Firebase. Utiliza un diseño multi-inquilino (multi-tenant) basado en `organizationId` para aislar los datos entre diferentes empresas. El sistema se organiza en módulos especializados (CRM, Inmobiliaria, Nómina, etc.) que comparten una base de componentes y servicios comunes.

```text
[main.jsx]
    |
[AuthContext] (Global State)
    |
[Modules] (Real Estate, CRM, ...)
    |
[Hooks] (useRealEstate, useCrm)
    |
[Firestore] (Data Layer)
```

### CRM Module

### Auth Context
- **Purpose:** Manejo de sesión, permisos y configuración de organización (moneda, branding).
- **Location:** `src/context/AuthContext.jsx`

## Data Flow

1. **Autenticación**: El usuario se loguea y el `AuthContext` obtiene los datos de la organización.
2. **Carga de Módulos**: Los hooks (ej: `useRealEstate`) se suscriben a colecciones de Firestore filtradas por `organizationId`.
3. **Interacción**: Los formularios validan datos con **Zod** antes de enviarlos a Firestore.
4. **Visualización**: Componentes premium con **Tailwind CSS** y animaciones renderizan los datos en tiempo real.
