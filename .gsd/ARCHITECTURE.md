## Overview

PKT ERP is a multi-tenant enterprise resource planning solution designed for small and medium businesses. It provides a modular approach, allowing clients to activate only the features they need (CRM, Real Estate, Projects, Inventory, Finance, Sales, Purchases, Calendar, etc.). The system is built as a single-page application (SPA) using React and Vite, backed by Firebase (Authentication and Firestore) for real-time data synchronization and multi-tenancy support.

## Data Flow

1. **Authentication:** User logs in via `src/modules/Login.jsx` using Firebase Auth.
2. **Context Initialization:** `AuthContext` listens to auth state changes, fetches user metadata from Firestore (`/users/{uid}`), determines the user's role and `organizationId`, and resolves their module subscriptions.
3. **Routing:** Users are redirected to `/admin/dashboard` or `/client/dashboard` based on their role.
4. **Data Fetching:** Feature modules use custom hooks (e.g., `useCrm.js`, `useSales.js`) to interact with Firestore. Queries are scoped to the user's `organizationId` (e.g., `organizations/{orgId}/invoices`).
5. **Real-time Updates:** Many modules use Firebase `onSnapshot` to automatically reflect database changes in the UI.
